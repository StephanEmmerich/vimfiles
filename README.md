Vim config
===========

Installation
------------

	$ git clone git@github.com:StephanEmmerich/vimfiles.git

	$ cd ~/.vim
	$ git submodule init
	$ git submodule update

	$ ln -s ~/.vim/vimrc ~/.vimrc
	$ ln -s ~/.vim/gvimrc ~/.gvimrc
